## üöÄ How to Use

### Backend Server

```bash
# Install dependencies
npm install

# Start server
npm start

# Or with auto-launch
npm start -- --auto-launch
```

### Mobile App

```bash
# Install dependencies
cd mobile
npm install --legacy-peer-deps

# Start app
npm start

# Or clear cache
npx expo start -c
```

### CLI Commands

```bash
# Start server
nano-cli start

# Auto-launch with Antigravity
nano-cli start --auto-launch

# Web mode (ngrok)
nano-cli start --web

# Show connection info
nano-cli info

# Configure
nano-cli config show
nano-cli config set password mypass

# SSL
nano-cli ssl status
nano-cli ssl generate
```
